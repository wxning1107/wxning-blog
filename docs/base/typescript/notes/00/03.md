## 枚举类型

定义枚举类型：
```typescript
enum HTTPStatus {
  OK, // 0
  NOT_FOUND, // 1
  INTERNAL_SERER_ERROR, // 2
}

console.log(HTTPStatus.NOT_FOUND) // 1
```
当然也可以赋初值：
```typescript
enum HTTPStatus {
    OK = 200, 
    NOT_FOUND = 404,
    INTERNAL_SERER_ERROR = 500, 
}

console.log(HTTPStatus.NOT_FOUND) // 404
```
获取枚举类型变量名：
```typescript
enum HTTPStatus {
    OK = 200, 
    NOT_FOUND = 404,
    INTERNAL_SERER_ERROR = 500, 
}

console.log(HTTPStatus[HTTPStatus.NOT_FOUND]) // "NOT_FOUND"
```
